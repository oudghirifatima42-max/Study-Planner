# Study Planner

Study Planner est une application Android développée en Java permettant aux étudiants de planifier et suivre leurs révisions avant les examens.

## Contenu du projet

- Code source Android Studio
- Rapport de stage
- Vidéo de démonstration

## Technologies utilisées

- Java
- XML
- SQLite
- Android Studio
